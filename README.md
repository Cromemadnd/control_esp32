# 引脚连接

B10 -> D16
B11 -> D17

继电器（5V供电）控制
D14 -> VIN

OLED（3.3V供电）控制
D21 -> SDA
D22 -> SCL

D12 -> DIN


# 语音模块部分


## 串口通信协议结构定义
1. 双帧头1 0xAA
2. 双帧头2 0x00
3. 功能码: 0x01(开关灯), 0x02(逆变电源控制), 0x03(屏幕控制)
4. 数据 1字节

> 注意数据帧长度定长, 4字节

## 逆变电源控制

> 中继器GPIO控制
- 开启逆变电源  0xAA 0x00 0x02 0x25

- 关闭逆变电源  0xAA 0x00 0x02 0x30

- 电量播报: rx :  0xAA, 0x00  0x02 <16进制电量(十进制100以下)> 

RX引脚

## 屏幕控制
- 屏幕开  0xAA 0x00 0x03 0x35
- 屏幕关  0xAA 0x00 0x03 0x40
RX 引脚


## 灯效控制

常亮 0xAA 0x00 0x04 0x60
呼吸 0xAA 0x00 0x04 0x60
SOS  0xAA 0x00 0x04 0x65
快闪 0xAA 0x00 0x04 0x70
