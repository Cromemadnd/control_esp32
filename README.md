# 引脚连接

串口通信 

stm32->esp32
B10 -> D16
B11 -> D17

语音模块
D4 -> PB5
D5 -> PB6

继电器（5V供电）控制
D14 -> VIN

OLED（3.3V供电）控制
D21 -> SDA
D22 -> SCL

55灯灯带（5V供电）控制
D12 -> DIN

# 串口通信协议结构定义
1. 双帧头1 0xAA
2. 双帧头2 0x00
3. 功能码: 0x01(开关灯), 0x02(逆变电源控制), 0x03(屏幕控制)
4. 数据 1字节

> 注意数据帧长度定长, 4字节

## 开关灯

开灯 0xAA 0x00 0x01 0x15
关灯 0xAA 0x00 0x01 0x20

## 逆变电源控制

开启逆变电源  0xAA 0x00 0x02 0x25
关闭逆变电源  0xAA 0x00 0x02 0x30

## 电量播报
- tx(语音模块 -> esp32):  0xAA 0x00 0x02 0x20
- rx(esp32 -> 语音模块):  0xAA 0x00 0x02 <16进制电量(十进制100以下)> 

## 屏幕控制
- 屏幕开  0xAA 0x00 0x03 0x35
- 屏幕关  0xAA 0x00 0x03 0x40

## 灯效控制

常亮 0xAA 0x00 0x04 0x60
呼吸 0xAA 0x00 0x04 0x62
SOS  0xAA 0x00 0x04 0x65
快闪 0xAA 0x00 0x04 0x70
